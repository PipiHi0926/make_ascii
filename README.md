# 圖像轉文字小幫手 🛠️ (Image to ACSII)

## ✔️ 介紹

這是一個建立圖像轉文字的小工具!!!  

只要上傳圖片，就能生成對應的文字、程式碼~ 

把乖乖、佛祖他們放在註解裡吧 😄

This is a tool to convert images to text!
Just upload an image, and it will generate the corresponding text or code.
Put them in your comments. 😄



## ✔️ 安裝與使用

### 1. 下載程式碼到本地
```=bash
git clone https://github.com/PipiHi0926/make_ascii.git
```

### 2. 進入專案目錄：切換到下載的專案目錄
```=bash
cd make_ascii
```
### 3. 安裝依賴
在專案目錄中，安裝所有必要的Python庫。您可以通過以下命令安裝所有依賴：
```=bash
pip install -r requirements.txt
```

### 4. 在專案目錄中，執行以下命令啟動Gradio應用：
```=bash
python make_ascii/mage_to_ascii.py
```
